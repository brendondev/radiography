Visão Geral

O sistema de aquisição de imagens radiográficas DigiXRay™ DR foi projetado para atender a uma grande faixa de aplicações em medicina veterinária. Nossos painéis DR possuem uma variedade de tamanhos e tipos oferecendo soluções específicas para cada aplicação radiográfica que o moderno médico veterinário necessite. Foram construidos para atender a norma IP64 no sentido de ingresso de poeira e liquidos. Através de uma construcao robusta utilizando laterais emborrachadas e um encapsulamento único feito com ligas de aluminio de alta dureza, nossos painéis são inerentemente robustos contra as condições adversas do ambiente. O sistema DigiXRay™ DR inclui software e computador de aquisição. O software de aquisição de imagens radiográficas DR oferece um suave e sistemático fluxo de trabalho. O processador de imagens permite surpreendente qualidade de imagem, foi desenvolvido para a otimização na vizualisação de orgãos especificos. Várias funções integradas como o guia de posicionamento radiográfico e sua operação intuitiva simplificam imensamente as rotinas do dia a dia. O visualizador integrado de imagens oferece ao usuário recursos de diagnostico dentro do software de aquisição. O sistema pode ser considerado uma estação de trabalho para diagnóstico com a opção de ser integrado a um sistema PACS (Picture Archiving and Communication System).
